Thank you for using Fcuk API, a Chrome extension built to help developers, testers, and non-coders efficiently test and validate the performance of APIs through load, spike, and stress testing. We are committed to protecting your privacy.

What Data We Collect

Fcuk API does not collect or store any personal data.

The extension captures API request metadata (like URLs, request headers, and response times) locally in your browser using temporary local storage. This data:

  • Is processed entirely within your browser
  • Is not permanently stored or sent to any server
  • Is used only to help visualize API performance

Data Sharing

We do not share any personal user data with third parties.

When you initiate a load test, only the API endpoint (not headers, tokens, or personal information) is passed to a secure AWS EC2 machine running a private k6 load testing service. This is strictly used to perform your requested performance analysis.

Ads and Analytics

We do not use ads or any analytics tracking. The only data shown in the extension UI are performance results from your load test, rendered in real time.

Purpose

Fcuk API is designed to support:

  • Developers and testers in load testing APIs quickly
  • Non-technical users in identifying API issues
  • Business owners in ensuring smoother website and app performance

Security

All performance tests are executed on a secure AWS EC2 instance with no data storage. Test endpoints are encrypted in transit and discarded after use.
